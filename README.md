# xiaoke
# 🚀 全能编程智能体 · 多智能体自动编程

一个全自动 AI 编程助手，支持多智能体协作、自动写代码、自动调试、前端、数据库、浏览器自动化、Docker、Git、联网搜索、长期记忆。

## 功能
- 多智能体架构（架构师 / 程序员 / 前端 / DBA / 自动化 / 运维 / 调试）
- 自动生成可运行项目
- 自动联网查资料
- 自动生成前端页面
- 自动生成数据库表
- 浏览器自动化（Playwright）
- 自动生成 Dockerfile
- 自动 Git 提交
- 自动调试修复 Bug
- 长期记忆
- 可编译 EXE 桌面软件
